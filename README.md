# ASMR Oasis

Одностраничный сайт с подборкой ASMR-контента: атмосферные аудио-сеансы, трансляции и подборки для релакса, сна и концентрации.

## Стек
- HTML5
- CSS3
- Vanilla JavaScript

## Локальный просмотр
Откройте файл `index.html` в браузере или поднимите простой сервер, например:

```bash
python3 -m http.server 8000
```

После запуска сайт будет доступен по адресу [http://localhost:8000](http://localhost:8000).
